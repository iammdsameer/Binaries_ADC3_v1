Features 

Git Clone

Usage: git clone [url]  

This command is used to obtain a repository from an existing URL.

Git Pull

Usage: git pull [Repository Link]  

This command fetches and merges changes on the remote server to your working directory.

Git Push

Usage: git push [variable name] master  

This command sends the committed changes of master branch to your remote repository.

Usage: git push [variable name] [branch]  

This command sends the branch commits to your remote repository.



Git Merge Conflict

Merge conflict error is created when files in local repo and remote do not match or same file in local repo and remote repo has got different changes.

For example:  when two developers set different values in same variable in the same file this conflict occurs.


Git Branch

Usage: git branch  

This command lists all the local branches in the current repository.

Usage: git branch [branch name]  

This command creates a new branch.


Git Commit

Usage: git commit -m “[ Type in the commit message]”  

This command records or snapshots the file permanently in the version history.
